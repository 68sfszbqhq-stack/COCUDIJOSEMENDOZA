# PLE - José Roberto Mendoza

Portafolio Digital de Entorno Personal de Aprendizaje (PLE).

## Descripción
Este proyecto visualiza la Matriz de Gestión de Información y Aprendizaje mediante un mapa interactivo y un diseño web moderno.

## Tecnologías
- HTML5
- Tailwind CSS
- Vanilla JavaScript
- FontAwesome

## Estructura
- `index.html`: Página principal con el mapa radial interactivo y la matriz.
